## 1. EMBER

**Best for:** Coffee shops, specialty cafes, pastry boutiques, artisan bakeries

**Mood:** Warm, intimate, artisanal—like entering a neighborhood roastery at dawn

**Primary Color Palette (Light Variant):**
- Background: `#FDF6F1` (warm cream)
- Surface: `#FFFFFF` (pure white)
- Text: `#1F1109` (deep charcoal)
- Muted: `#8B7355` (soft brown)
- Border: `#D9CCC0` (light taupe)
- Accent: `#A34C24` (terracotta rust)
- Accent-Contrast: `#FFFFFF` (white text on rust)

**Alternate Palette (Dark Variant):**
- Background: `#2A1810`
- Surface: `#3D2817`
- Text: `#F5EFE8`
- Muted: `#A8927D`
- Border: `#4F3D2F`
- Accent: `#E8885C` (warm apricot)
- Accent-Contrast: `#1F1109` (dark text on apricot)

**Font Pairing:** Cormorant Garamond (display) + Inter (body)

**Layout & Section Concept:**
- **Hero:** Full-width hero image (coffee being poured, latte art, pastry close-up) overlaid with a diagonal typography banner saying business name + tagline. Not centered; positioned bottom-right on image.
- **Sections in order:** Hours strip (horizontal, sticky top), Coffee/Menu grid with hover-state image swap, About (text-left, image-right), Instagram feed grid, Contact + Map (simple inline), Footer
- **Distinctive feature:** Diagonal angled section breaks between hero and menu; typography that uses hanging letters/slight kerning play

**Imagery Direction:** Close-up food/drink photography; steam, wood grain, pour-overs, textured cups; warm golden-hour light; hands in action

**Signature Element:** Angled diagonal divider bands between sections that echo the angle of a coffee pour

**Imagery Direction:** Editorial fashion photography, styled product shots, texture close-ups, mood imagery, minimalist still-life, sophisticated color palette

**Signature Element:** Asymmetrical layout with large hero image bleeding to edge; thin copper rule accents between sections; serif display type positioning text artfully alongside imagery

## Design Principles Across All Templates

1. **No AI clichés:** None uses the cream + high-contrast serif + terracotta pattern exclusively. Each has a distinct palette, typography pairing, and layout approach.
2. **Real hierarchy:** Each template prioritizes different information (Ember: menu; Sapphire: credentials; Luminous: portfolio).
3. **Mood varies:** Light/dark balance, warm/cool balance, minimal/bold balance, and classic/trendy span across all 15.
4. **Distinctive elements:** Each is remembered by a signature feature (Ember's diagonal dividers, Nocturne's gold borders, Atelier's asymmetrical layout, etc.).
5. **Photography direction clear:** Every template knows what imagery works best, avoiding generic stock photos.
